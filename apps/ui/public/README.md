# Public Assets

This directory contains static assets that are served by Next.js.

## Logo

Place your logo image file here as `logo.png`.

The logo should be:
- Format: PNG (recommended) or SVG
- Recommended dimensions: At least 240x240px for high-quality display
- The logo will be automatically resized for different contexts:
  - Small (sm): 40x40px (navigation bar)
  - Medium (md): 60x60px (default)
  - Large (lg): 120x120px (login page)
